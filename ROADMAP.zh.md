# 路线图

## v0.3.0 — 性能与稳定性
- [ ] SSH 会话连接池
- [ ] 并行传输评分（减少初始连接时间）
- [ ] 持久化状态 / 崩溃恢复
- [ ] 传输评分基准测试套件
- [ ] 内存优化（减少热路径中的分配）

## v0.4.0 — Web UI
- [ ] Web 仪表盘（React/Vue + Go API 服务器）
- [ ] WebSocket 实时更新
- [ ] 远程服务器浏览器
- [ ] 隧道配置编辑器（Web 表单）
- [ ] 移动端响应式布局

## v0.5.0 — 高级网络
- [ ] TUN 绑定（将多个隧道聚合为一个接口）
- [ ] 逐包负载均衡
- [ ] NAT 穿透 / STUN 支持
- [ ] 仅 IPv6 模式
- [ ] 隧道 DNS 的 DNS-over-HTTPS

## v0.6.0 — 企业级
- [ ] 多用户 RBAC
- [ ] 审计日志（所有命令 + 配置变更）
- [ ] LDAP/OIDC 认证
- [ ] SLA 报告（正常运行时间 %、故障转移时间）
- [ ] Terraform 提供者用于隧道配置

## v1.0.0 — 稳定版本
- [ ] 测试覆盖率 90%+
- [ ] 所有传输解析器的 Fuzz 测试
- [ ] 正式安全审计
- [ ] 签名发布（cosign + SLSA）
- [ ] LTS 支持策略
