# نقشه راه

## v0.3.0 — عملکرد و پایداری
- [ ] استخر اتصال برای نشست‌های SSH
- [ ] امتیازدهی موازی ترنسپورت‌ها (کاهش زمان اتصال اولیه)
- [ ] وضعیت پایدار / بازیابی از خرابی

## v0.4.0 — رابط وب
- [ ] داشبورد وب (React/Vue + Go API server)
- [ ] به‌روزرسانی‌های بلادرنگ WebSocket
- [ ] مرورگر سرورهای راه دور
- [ ] ویرایشگر تنظیمات تونل (فرم وب)
- [ ] طراحی واکنش‌گرا برای موبایل

## v0.5.0 — شبکه پیشرفته
- [ ] TUN bonding (ترکیب چندین تونل در یک رابط)
- [ ] توزیع بار بر اساس بسته
- [ ] عبور NAT / پشتیبانی STUN
- [ ] حالت فقط IPv6
- [ ] DNS-over-HTTPS برای DNS تونل

## v0.6.0 — سازمانی
- [ ] RBAC چند کاربره
- [ ] لاگ‌گیری حسابرسی (تمام دستورات + تغییرات تنظیمات)
- [ ] احراز هویت LDAP/OIDC
- [ ] گزارش‌دهی SLA (درصد آپتایم، زمان فیلور)
- [ ] ارائه‌دهنده Terraform برای راه‌اندازی تونل

## v1.0.0 — نسخه پایدار
