# Дорожная карта

## v0.3.0 — Производительность и стабильность
- [ ] Пул соединений для SSH сессий
- [ ] Параллельное тестирование транспортов (сокращение времени начального подключения)
- [ ] Устойчивое состояние / восстановление после сбоя
- [ ] Бенчмарк-набор для тестирования транспортов
- [ ] Оптимизация памяти (сокращение аллокаций в горячих путях)

## v0.4.0 — Веб-интерфейс
- [ ] Веб-дашборд (React/Vue + Go API сервер)
- [ ] Обновления в реальном времени через WebSocket
- [ ] Браузер удалённых серверов
- [ ] Редактор конфигурации туннелей (веб-форма)
- [ ] Адаптивный дизайн для мобильных устройств

## v0.5.0 — Продвинутые сети
- [ ] TUN-биндинг (агрегация нескольких туннелей в один интерфейс)
- [ ] Балансировка нагрузки по пакетам
- [ ] NAT traversal / поддержка STUN
- [ ] Режим только IPv6
- [ ] DNS-over-HTTPS для DNS туннелей

## v0.6.0 — Предприятие
- [ ] Мультипользовательский RBAC
- [ ] Аудит-логирование (все команды + изменения конфигурации)
- [ ] LDAP/OIDC аутентификация
- [ ] SLA-отчётность (время работы %, время переключения)
- [ ] Terraform провайдер для провижининга туннелей

## v1.0.0 — Стабильный релиз
- [ ] Покрытие тестами 90%+
- [ ] Fuzz-тестирование для всех парсеров транспортов
- [ ] Формальный аудит безопасности
- [ ] Подписанные релизы (cosign + SLSA)
- [ ] Политика поддержки LTS
