## Before Asking

1. Check the [FAQ](FAQ.md)
2. Search [existing issues](https://github.com/nyxora-network/nyxora/issues)
3. Search [discussions](https://github.com/nyxora-network/nyxora/discussions)
4. Run `nyxora --help`
