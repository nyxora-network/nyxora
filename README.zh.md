## рҹҡҖ еҝ«йҖҹејҖе§Ӣ

```bash
# 1. й…ҚзҪ®е№¶жЈҖжҹҘдҫқиө–
nyxora install

# 2. иҝһжҺҘеҲ°иҝңзЁӢжңҚеҠЎеҷЁ
nyxora connect 192.168.1.100 --user root --password your_password

# 3. еҗҜеҠЁдәӨдә’ејҸ TUI
nyxora tui

# 4. е®һж—¶зӣ‘жҺ§д»ӘиЎЁжқҝ
nyxora dashboard
```

### иҝһжҺҘйҖүйЎ№

```bash
nyxora connect <host> [options]

йҖүйЎ№пјҡ
  --user, -u <name>       SSH з”ЁжҲ·еҗҚпјҲй»ҳи®Өпјҡrootпјү
  --port, -p <port>       SSH з«ҜеҸЈпјҲй»ҳи®Өпјҡ22пјү
  --password <pass>       SSH еҜҶз Ғ
  --mode <mode>           жңҚеҠЎеҷЁжЁЎејҸпјҡfull, lite, minimal
  --transports <list>     дј иҫ“еҲ—иЎЁпјҲиҰҶзӣ– modeпјү
  --ports <pairs>         з«ҜеҸЈиҰҶзӣ–пјҡwg=51820,ss=8388,...
```

#### жңҚеҠЎеҷЁжЁЎејҸ

| жЁЎејҸ | дј иҫ“ | жүҖйңҖеҶ…еӯҳ |
|------|------|----------|
| `full` | е…ЁйғЁ 11 з§Қйҡ§йҒ“ | 2GB+ |
| `lite` | иҪ»йҮҸйҖүжӢ© | 512MBвҖ“2GB |
| `minimal` | SSH + Shadowsocks д»… | < 512MB |
