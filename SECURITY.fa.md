## گزارش آسیب‌پذیری

امنیت برای ما بسیار مهم است. اگر آسیب‌پذیری امنیتی در NYXORA کشف کردید،
**لطفاً** یک issue عمومی باز نکنید.

به جای آن، گزارش خود را به کانال تلگرام ما ارسال کنید: [@NyxoraCore](https://t.me/NyxoraCore)

ما ظرف ۴۸ ساعت تأیید دریافت ارسال می‌کنیم و ظرف ۷ روز پاسخ دقیق ارائه خواهیم داد.

### اطلاعات مورد نیاز

- توضیحات آسیب‌پذیری
- مراحل بازتولید
- نسخه‌های تحت تأثیر
- تأثیر بالقوه
- پیشنهاد رفع (در صورت وجود)

## بهترین شیوه‌های امنیتی

هنگام استفاده از NYXORA:

1. **از رمزهای عبور قوی SSH** یا احراز هویت مبتنی بر کلید استفاده کنید
2. **رمزهای عبور پیش‌فرض را تغییر دهید** — NYXORA به صورت خودکار رمزهای عبور برای Shadowsocks، Rathole، Hysteria، Backhaul و IPsec تولید می‌کند، اما برای استفاده در محیط تولید باید آن‌ها را به صورت صریح از طریق متغیرهای محیطی تنظیم کنید
3. **قوانین فایروال** — هم `nyxora install` و هم فرآیند اتصال قوانین iptables را برای محدود کردن دسترسی تونل تنظیم می‌کنند
4. **به‌روزرسانی منظم** — `nyxora update` اجرا کنید یا صفحه نسخه‌های GitHub را بررسی کنید
5. **اجرای غیر root** در صورت امکان (اگرچه راه‌اندازی تونل برای WireGuard و iptables به root نیاز دارد)
