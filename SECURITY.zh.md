## 报告漏洞

我们非常重视安全。如果您在 NYXORA 中发现安全漏洞，请**不要**打开公开 issue。

请通过我们的 Telegram 频道发送报告：[@NyxoraCore](https://t.me/NyxoraCore)

我们将在 48 小时内确认收到，并在 7 天内提供详细回复。

### 需要包含的信息

- 漏洞描述
- 重现步骤
- 受影响的版本
- 潜在影响
- 建议的修复方案（如果有）

## 安全最佳实践

使用 NYXORA 时：

1. **使用强 SSH 密码**或基于密钥的认证
2. **更改默认密码** — NYXORA 为 Shadowsocks、Rathole、Hysteria、Backhaul 和 IPsec 自动生成密码，但生产环境应通过环境变量显式设置
3. **防火墙规则** — `nyxora install` 和连接过程会设置 iptables 规则以限制隧道访问
4. **定期更新** — 运行 `nyxora update` 或检查 GitHub releases 页面
5. **尽可能以非 root 用户运行**（尽管隧道设置需要 root 权限用于 WireGuard 和 iptables）
